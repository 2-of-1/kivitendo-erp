6.6 Anlegen der Authentifizierungsdatenbank

Nachdem alle Einstellungen in config/authentication.pl vorgenommen wurden, muss Lx-Office die Authentifizierungsdatenbank anlegen. Dieses geschieht automatisch, wenn Sie sich im Administrationsmodul anmelden, das unter der folgenden URL erreichbar sein sollte:

http://localhost/lx-erp/admin.pl
