6 Benutzer und Datenbanken anlegen

Nach der Installation müssen Benutzer und Datenbanken angelegt werden. Dieses geschieht im Administrationsmenü, dass Sie unter folgender URL finden:

http://localhost/lx-erp/admin.pl

Zuerst muss eine Datenbank angelegt werden, anschließend ein Benutzer. Verwenden Sie für den Datenbankzugriff den eben angelegten Benutzer “lxoffice”.

Wenn Sie für die Lx-Office-Installation nicht den europäischen Schriftsatz ISO-8859-15 sondern UTF-8 (Unicode) benutzen wollen, so müssen Sie vor dem Anlegen der Datenbank in der Datei “lx-erp.conf” die Variable $dbcharset auf den Wert “UTF-8” setzen. Zusätzlich muss beim Anlegen der Datenbank “UTF-8 Unicode” als Schriftsatz ausgewählt werden.
